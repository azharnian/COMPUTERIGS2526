Solusi Otomatisasi Kebun Anda

Smart Garden Palembang adalah penyedia solusi dan API untuk sistem monitoring kebun pintar di wilayah tropis, terkhusus di kota Palembang.

Lihat Dokumentasi API

Fitur Unggulan

  • Sensor kelembaban dan suhu real-time
  • Integrasi dengan sistem irigasi otomatis
  • Dashboard web untuk monitoring kebun
  • API terbuka untuk developer

API Kami

Kami menyediakan RESTful API yang memungkinkan integrasi data kebun dengan aplikasi Anda, baik untuk analisis, otomasi, atau IoT.

GET https://api.smartgardenpalembang.id/v1/sensors
